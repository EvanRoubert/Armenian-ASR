# Armenian ASR

This project aims to develop a state-of-the-art Automatic Speech Recognition (ASR) model for the Armenian language.

## Status

🚧 Research in progress.

A large-scale speech dataset has already been collected and prepared. The current stage focuses on model training and evaluation.

The source code, model checkpoints, and documentation will be released after the initial research phase.

## Goals

* High-accuracy Armenian speech recognition
* Robust performance across different speakers and recording conditions
* Open-source release after the research milestone is completed
